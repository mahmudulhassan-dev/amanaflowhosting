# Component List (Epic 2)

## `Button`
- Variants: `primary`, `secondary`, `outline`, `ghost`, `danger`.
- Sizes: `sm`, `md`, `lg`, `xl`.
- Features: Icon support, loading state glow.

## `Badge`
- Variants: `success` (Emerald), `warning` (Amber), `neutral` (Slate).
- Features: Dot indicator for live status.

## `SectionWrapper`
- Standardizes container padding and max-width (`max-w-7xl`).

## `Card` & `IconCard`
- Used for domains, hosting features, and pricing.
- Hover states with glowing borders in deep navy.
