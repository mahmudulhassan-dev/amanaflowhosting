# Design System

AmanaFlow utilizes a premium hybrid enterprise/SaaS design system.

## Tokens Reference
- Primary: Royal Blue (`#3b82f6` to `#1e3a8a`)
- Accent: Cyan (`#06b6d4` to `#0e7490`)
- Surfaces: Deep Navy/Slate (`#030712` to `#f8fafc`)
- Spacing: 8pt grid (`p-2`, `p-4`, `p-8`)
- Shadows: Premium glows and sharp elevations.

## Base UI Components
- **Button**: Glassmorphism and solid variants.
- **Badge**: Sharp edges with glowing dots.
- **Card**: Bordered, subtle shadow, dark mode optimized.
