## Speed is a Ranking Factor
Since 2010 (and updated in 2021 with the Page Experience Update), Google has explicitly used site speed as a ranking signal. But it's not just about "fast." It's about **Core Web Vitals (CWV)**.

### The Metrics that Matter

1. **TTFB (Time to First Byte)**: How long it takes for a server to respond. **Goal: < 200ms**.
2. **LCP (Largest Contentful Paint)**: How long it takes for the main content to load. **Goal: < 2.5s**.
3. **FID (First Input Delay)**: How fast the page responds to a user's click. **Goal: < 100ms**.
4. **CLS (Cumulative Layout Shift)**: Does the page jump around while loading? **Goal: < 0.1**.
